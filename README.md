# Purple
A Hugo Theme.

## Installation
```console
git submodule add https://github.com/ianmuchina/purple themes/purple
```

## Configuration

`config.yml example`
```yaml
theme: purple

params:
  author: #
  description: #
  twitter: #twitter Username
  github: #github Username
  email: # mailto link
  og_image: #Opengraph Image
```
